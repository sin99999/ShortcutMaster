# ShortcutMaster

いま前面で使っているアプリに合わせて「今使えるキーボードショートカット」を画面右下の小さなチップから確認し、クリック 1 回でそのまま実行できる Windows 用の常駐ツールです。

## 特長

- 画面右下に極小の 1 行チップとして常駐します
- チップをクリックすると、上方向にショートカット一覧が展開します
- 前面アプリ（Windows / Cursor / Excel）に応じて一覧が自動で切り替わります
- 一覧の項目をクリックすると、直前まで操作していたアプリへそのショートカットを送信します
- パネルはフォーカスを奪わない設計のため、コピーや貼り付けなども元のアプリに正しく届きます
- よく使う項目ほど「おすすめ」の上位に表示されます（学習データは端末内のみ）
- AI・クラウド接続は使用しません。完全にローカルで動作します

## 動作環境

- Windows 10 / 11（64bit）
- ビルド・実行には **.NET 10 SDK** が必要です（リポジトリの `global.json` で SDK 10 を指定）

## ビルドと実行

```powershell
dotnet build ShortcutMaster.slnx -c Release
dotnet run --project src/ShortcutMaster
```

テストの実行:

```powershell
dotnet test ShortcutMaster.slnx
```

## 使い方

1. 起動すると画面右下にチップ（「◯◯ のショートカット」）が表示されます
2. チップをクリックすると一覧が開きます
3. 実行したい項目をクリックすると、直前のアプリにキーが送信されます
4. 一覧を閉じるには、チップの再クリックまたは右上の ✕ を使います
5. 終了するには、タスクトレイのアイコンを右クリックして「終了」を選択します

## 制限事項

- 管理者権限で動作しているアプリには送信できません（送信前に案内を表示します）
- Ctrl+Alt+Del などシステム専用の操作は「表示のみ」です
- 日本語入力（IME）で変換中は送信しません（案内を表示します）
- 現在のバージョンでは、チップはプライマリモニタの右下に固定です
- ロック（Win+L）・終了（Alt+F4）・完全削除（Shift+Delete）など破壊的操作は「表示のみ」です
- ゲームなど入力を専有するアプリは対象外です
